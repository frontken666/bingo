## 🚀 使用方式

### 1. 設定環境變數

編輯 `.env.local` 文件：

```env
GROQ_API_KEY=你的_groq_api_金鑰
```

### 2. 啟動開發服務器

```bash
npm run dev
```

### 3. 瀏覽器訪問

```
http://localhost:3000
```

## 📱 功能說明

### 投注參數設定
- **幾星**：3-6 星選擇
- **幾倍**：1-10 倍投注
- **幾期**：1-10 期連續
- **幾注**：1-10 注數量

### AI 智慧分析策略
1. **熱門策略** 🔥 - 高頻號碼分析
2. **冷門策略** ❄️ - 低頻號碼推薦
3. **連號策略** ⚡ - 連續號碼模式
4. **混搭策略** ✨ - 平衡熱冷號碼
5. **AI 建議** 🧠 - 智慧綜合分析

### 資料視覺化
- **號碼球全覽**：1-80 號碼熱度圖
- **頻率分析圖表**：長條圖統計
- **熱門/冷門排行**：TOP 10 榜單
- **歷史記錄**：最近 20 期開獎

## 🎯 使用技巧

1. **選擇分析範圍**：建議使用 20-50 期資料，較為準確
2. **策略搭配**：可先用 AI 建議，再比對熱門/冷門策略
3. **號碼標註**：AI 推薦的號碼會在歷史記錄中以紫色高亮顯示
4. **頻率觀察**：號碼球的顏色深淺代表出現頻率
