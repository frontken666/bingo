# 兌獎系統使用說明

## 功能介绍

兌獎系統可以自動對比您的投注號碼與最新開獎結果，計算中獎金額和獲利情況。

## 使用方法

### 1. 輸入投注號碼

在文字框中輸入您的投注號碼，格式如下：

```
4,11,18,25
5,12,19,26
7,14,20,27
8,15,21,29
11,16,22,30
12,18,24,32
13,19,25,33
14,20,26,35
15,21,27,36
16,22,37,64
```

**格式要求：**
- 每行代表一注
- 號碼之間用逗號分隔
- 支援2-10星（每行2-10個號碼）

### 2. 點擊「開始兌獎」

系統會自動：
- 對比最新開獎結果
- 計算每注的中獎情況
- 顯示中獎號碼（綠色標記）
- 計算中獎金額

### 3. 查看結果

系統會顯示：

#### 中獎列表
- 只顯示有中獎的注
- 每注顯示：
  - 注數編號
  - 選擇的號碼（中獎的顯示綠色）
  - 中了幾個號碼
  - 中獎金額

#### 統計資訊
- **總成本**：所有注的投注金額總和
  - 計算公式：注數 × $25 × 倍數
- **中獎總額**：所有中獎金額的總和
- **獲利**：中獎總額 - 總成本
  - 正數：盈利（黃色顯示）
  - 負數：虧損（紅色顯示）

## 範例

假設輸入10注3星號碼，倍數為1倍：

```
4,11,18,25    ← 假設中3個
5,12,19,26    ← 沒中
7,14,20,27    ← 假設中2個
```

結果：
- **總成本**：10注 × $25 × 1倍 = $250
- **第1注**：4星中3個 → $2,000
- **第3注**：4星中2個 → $20
- **中獎總額**：$2,020
- **獲利**：+$1,770（盈利）

## 注意事項

1. **倍數設定**：
   - 需要在頁面上方的「投注參數設定」中先設定倍數
   - 中獎金額會根據倍數自動計算

2. **最新開獎**：
   - 系統自動使用最新一期的開獎結果
   - 確保資料已載入後再進行兌獎

3. **號碼格式**：
   - 號碼範圍：1-80
   - 無效的號碼會被自動忽略
   - 每行可以有不同的星數

4. **清空功能**：
   - 點擊「清空」按鈕可清除所有輸入和結果
   - 重新開始兌獎

## 獎金表（已加倍）

系統使用的是官方獎金表的2倍金額，具體獎金請參考成本分析區塊。
