# 🚀 快速启动指南

## 前置准备

### 1. 获取 Groq API Key

访问 [Groq Console](https://console.groq.com/) 注册并获取免费的 API Key。

### 2. 配置环境变量

打开项目根目录的 `.env.local` 文件，将您的 API Key 填入：

```
GROQ_API_KEY=gsk_your_actual_api_key_here
```

## 启动步骤

### 方法 1: 使用 npm

```bash
# 确保已安装依赖（如果还没有运行过）
npm install

# 启动开发服务器
npm run dev
```

### 方法 2: 使用 PowerShell 脚本

双击运行 `start.ps1` 或在 PowerShell 中执行：

```powershell
.\start.ps1
```

## 访问应用

启动成功后，在浏览器中打开：

**http://localhost:3000**

## 功能说明

### 1. 投注参数设置
- **几星**: 选择 3-6 星（推荐号码数量）
- **几倍**: 选择 1-10 倍（投注倍数）
- **几期**: 选择 1-10 期（连续投注期数）
- **几注**: 选择 1-10 注（投注注数）

### 2. AI 分析策略
- **热门策略** 🔥: 分析高频出现的号码
- **冷门策略** ❄️: 关注低频出现的号码
- **连号策略** ⚡: 识别连续号码规律
- **混搭策略** ✨: 平衡热门与冷门号码
- **AI 建议** 🧠: AI 综合智能分析

### 3. 数据可视化
- **号码球图表**: 1-80 号码出现频率统计
- **热门排行榜**: 出现次数最多的 TOP 10
- **冷门排行榜**: 出现次数最少的 TOP 10
- **历史记录**: 最近开奖结果（AI 推荐号码会特别标注）

## 常见问题

### Q1: 页面显示 "请设置 GROQ_API_KEY 环境变量"
**A**: 请检查 `.env.local` 文件是否正确配置了 API Key，并重启开发服务器。

### Q2: 数据来源显示 "模拟数据"
**A**: 官方 API 可能暂时无法访问，系统会自动使用模拟数据作为演示。功能完全正常。

### Q3: AI 分析速度较慢
**A**: Groq API 的免费版可能有速率限制，请耐心等待 5-10 秒。
